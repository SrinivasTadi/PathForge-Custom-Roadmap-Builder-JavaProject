# PathForge – Interactive Learning Roadmap Builder

## Overview

PathForge is a full-stack web application that enables students and learners to create, manage, and visualize personalized learning roadmaps. The system provides an interactive interface where users can create learning topics as nodes, connect them using visual relationships, track progress, and save their roadmap data permanently.

The project is developed using Spring Boot, MySQL, JavaScript, HTML, CSS, and jsPlumb.

---

## Features

- User Registration and Login
- Personalized Roadmap Creation
- Dynamic Node Management
- Visual Topic Connections using jsPlumb
- Progress Tracking
  - To Learn
  - In Progress
  - Completed
- Notes Management
- Roadmap Persistence
- Responsive Dashboard Interface
- MySQL Database Integration
